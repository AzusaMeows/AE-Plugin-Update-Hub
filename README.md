# AE Plugin Update Hub

Public update source for the After Effects panel “超级插件管理中心”.

## Binary distribution

Binary plugin packages are not distributed from this public repository.

## Update Sources

The panel self-update flow reads raw text files from Gitee first, then GitHub.

## AE Panel Update

Open the panel inside After Effects:

```text
Window > Extensions > 超级插件管理中心
```

Use the settings page to check for panel updates.
